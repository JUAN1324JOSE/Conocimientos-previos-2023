  • ETIQUETAS DE CABECERA

    ¿Qué son las etiquetas head?: La HEAD es la primera de las dos partes en que se estructura un documento HTML. En la HEAD reside información acerca del documento, y generalmente no se ve cuando se navega por él.

  • ETIQUETAS DE AGRUPACION

    Las siguientes etiquetas se utilizan para agrupar información:

    1. DIV: Capa o división utilizado para agrupar varias etiquetas HTML.
    2. P: Define un párrafo de texto (con sus etiquetas HTML para texto).
    3. MAIN: Contenedor para englobar la parte principal de la página.
    4. OL: Crea una lista numerada (con orden).
    5. UL: Crea una lista donde el orden no importa.
    6. Contiene uno de los ítems de una lista numerada o sin numerar.

  • ETIQUETAS DE MULTIMEDIA

    1. IMG: El elemento incrustado de imagen. El elemento de imagen HTML img representa una imagen en el documento
    2. VIDEO: se utiliza para incrustar vídeos en un documento HTML o XHTML.
    3. AUDIO: El elemento audio se usa para insertar contenido de audio en un documento HTML o XHTML.
    4. TRACK: Las pistas de texto especificadas con la etiqueta track podrían incluir subtítulos, títulos, descripciones, capítulos y metadatos.

  • ETIQUETAS DE TEXTO

    1. a: crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.
    2. strong: El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.
    3. code: se utiliza para mostrar texto que debe ser interpretado como código informático.
    4. sup-sub: se utiliza para representar texto en formato de subíndice dentro de un documento HTML.
